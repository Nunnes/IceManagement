- instalei o mysql-client e mysql-server (escolhi uma pass para o root)

- Entrei como root

     $ mysql -h localhost -u root -p

- Criei um novo utilizador (com todos os privilegios - like a boss)

     mysql> create user 'ice_manager'@'localhost' IDENTIFIED BY 'ice';
     mysql> GRANT ALL ON *.* TO 'ice_manager'@'localhost';
     mysql> exit

- Entrei no novo utilizador

     $ mysql -h localhost -u ice_manager -p

- Criei e entrei numa nova base de dados

    mysql> create database ice_db;
    mysql> use ice_db;
